Example For Storing Device Info


Device Name - can be whatever name you choose. eg. Sony Xperia Mobile
Purchase Date - date a device was purchased eg. 12/12/2012
Expiry Date - date warranty of the device expires eg. 12/6/2013
Notes - can contain comments for personal reference
Calendar Alert - if checked, creates an event in Google Calendar, to indicate the completion of warranty period

Example For Storing Service Info


Device Name - device / gadget / item that was serviced eg. Bajaj Pulsar
Purchase Date - date it was serviced eg. 12/12/2012
Expiry Date - date it is next due for service eg. 12/3/2013
Notes - can contain comments for personal reference. eg. Tyres changed etc.
Calendar Alert - if checked, creates an event in Google Calendar for the next service due date
